# Scheme Interpreter

한 줄씩 입력된 코드를 읽어 Evaluation 후 그 결과 출력

## 주요 기능

* 사칙연산
* 리스트 선언
* 값 할당
* 간단한 함수 구현
* 할당된 메모리 부족 시 Garbage Collector 실행

## 사용 예

'>' 에 코드 입력 시 ']' 에 결과 출력
